Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

You add a declaration for the position property with the value then you add the declaration of the other value and then for the top and specify a zero value.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

Add a declaration for the position property with the value of relative. Add another declaration for thr top property and specify a value of 90 px. Add another declarationfor thr left property with a value of 30 px.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

Add a declaration for the position property with a value of fixed. Add another declaration for the top property and specify a value of 90 px and then do that for the right of 90px

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

Add a declaration for the position property with a value of absolute. Add a declaration for the right property and set to 90px. Do the same for the bottom and set to 0
